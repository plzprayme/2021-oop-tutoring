# 2021-2 HNU튜터링 [객체지향 프로그래밍]



## 🎊 목표

1. Java의 기본 문법을 공부한다.
2. Kotlin의 기본 문법을 공부한다.
3. Java와 Kotlin을 비교하며 공부한다.
4. 코딩 테스트 문제를 풀며 공부한 내용을 체화시킨다.
5. 이번 학기 객체지향 프로그래밍 A+을 받는다.

## 👨‍🏫 수업 진행 방식

1. 멘토가 강의를 한다.

2. 강의를 따라 IntelliJ에서 실습한다.

3. 모르는 것들 질문한다.

4. 다음 수업까지 프로그래머스, 백준 문제 풀어본다.

5. 풀면서 모르는 것들 질문한다.
